# kiri

[![Go Version](https://img.shields.io/badge/Go-1.22+-00ADD8?style=flat-square&logo=go)](https://go.dev)

[Read in English](README.md) | [Русский]

**kiri** — это минималистичный, умный локальный TUI-помощник (Terminal User Interface) для ухода за вашими растениями. Программа плавно рассчитывает «уровни жажды», напоминает о поливе, ведет подробный журнал ухода и динамически подстраивается под погоду и сезон для уличных растений.

Все данные хранятся исключительно локально на вашем компьютере в легковесной базе SQLite.

## Особенности

- **Полноценный интерактивный TUI:** Вкладки, сетка карточек с динамическим изменением цвета в зависимости от состояния, календарь задач на месяц и полная поддержка мыши и клавиатуры.
- **Динамическая модель «жажды»:** Уровень воды уменьшается плавно со временем, а не резкими скачками раз в сутки. Для уличных растений модель учитывает сезонные изменения, температуру воздуха и осадки (данные автоматически подтягиваются в фоновом режиме из Open-Meteo API).
- **Календарь и Журнал:** Планируйте график полива заранее с помощью встроенного календаря задач и просматривайте историю действий в хронологическом журнале ухода.
- **UNIX-way и Скриптуемость:** Наличие No-TUI флагов (`--status`, `--summary`, `--json`) позволяет выводить текстовый статус или сырой JSON напрямую в ваши кастомные скрипты и системные панели (`Waybar`, `Polybar`, `i3blocks`, `tmux`).
- **Прозрачный режим (**`--transparent`**):** Отключает отрисовку фонов интерфейса, позволяя приложению идеально вписаться в кастомную тему вашего терминала или тайлового оконного менеджера.
- **Легковесность и Автономность:** Работает на базе быстрой локальной БД SQLite. Компилируется в один бинарник без внешних зависимостей во время выполнения (runtime).

## Быстрый старт



### Требования

- **Go 1.22** или выше.
- Любой современный эмулятор терминала (Alacritty, Ghostty, Kitty, GNOME Terminal и др.).



### Запуск из исходников

```bash
git clone https://github.com/freislot/kiri.git
cd kiri
go run .
```



### Компиляция бинарника

```bash
CGO_ENABLED=1 go build -o kiri .
./kiri
```

> **Совет для быстрого старта:** При самом первом запуске `kiri` автоматически заполнит базу данных **9 демо-растениями** с разными уровнями жажды. Вы сможете сразу оценить работу интерфейса, графиков и календаря без ручной настройки.

---



### Базовое управление

- `Tab` / `Shift+Tab` или `1`-`5` — Переключение между вкладками
- `↑ / ↓` или `j / k` — Навигация по спискам
- `Enter` — Посмотреть детали растения
- `q` / `Ctrl+C` — Выход из приложения

## Установка



### Прямая загрузка (Pre-compiled Binaries)

Вы можете скачать готовый бинарник или `.deb` пакет для вашего дистрибутива на странице **Releases**.

Для установки `.deb` пакета в Ubuntu/Debian:

```bash
sudo apt install ./kiri_0.1.0_linux_amd64.deb
```

### macOS

> **Примечание:** в будущем программа появится в homebrew. 

Если вы скачали архив с бинарником вручную, операционная система автоматически вешает на него флаг карантина. Чтобы успешно запустить утилиту в терминале, распакуйте архив, перейдите в папку с файлом и снимите ограничения безопасности следующими командами:

```bash
chmod +x ./kiri
xattr -d com.apple.quarantine ./kiri
```

### Arch Linux (AUR)

> **Примечание:** Официальные пакеты в AUR (`kiri` / `kiri-bin`) появятся в будущих обновлениях. 

Пока что пользователи Arch могут легко установить `kiri` из исходников с помощью `go` CLI или предоставленного `Makefile`. Программа автоматически интегрируется в ваш системный путь (PATH).

### Через Go CLI

```bash
go install github.com/freislot/kiri@latest
```



### Сборка через Makefile

В проекте есть стандартный GNU Makefile для локальной сборки и обслуживания:

```bash
make            # Скомпилировать бинарник
sudo make install # Установить бинарник и man-страницу в систему
sudo make uninstall # Полностью удалить программу из системы
```

## Флаги запуска

Вы можете использовать `kiri` в качестве легковесной утилиты для скриптов, не открывая интерактивный графический интерфейс. Справка по флагам: `kiri -h`


| Флаг            | Описание                                                                                                     |
| --------------- | ------------------------------------------------------------------------------------------------------------ |
| `--version`     | Показать версию программы (`v0.1.0`).                                                                        |
| `--status`      | Вывести форматированную таблицу статуса растений прямо в консоль.                                            |
| `--summary`     | Вывести одну строку, оптимизированную для статус-баров (например, `💧 3 требуют полива!` или `🌿 kiri: ok`). |
| `--json`        | Вывести полный машиночитаемый JSON-дамп всех растений для интеграций.                                        |
| `--transparent` | Принудительно запустить TUI в прозрачном режиме (без фоновых заливок).                                       |

## Пути к файлам и XDG-стандарт

Приложение строго следует спецификации **XDG Base Directory**. Все ваши данные находятся в безопасности и изолированы внутри домашней директории:


| Файл                         | Путь в системе                                     |
| ---------------------------- | -------------------------------------------------- |
| **База данных SQLite**       | `~/.config/kiri/data.db`                           |
| **Конфигурация TOML**        | `~/.config/kiri/settings.toml`                     |
| **Резервные копии (Бэкапы)** | `~/.config/kiri/data.db.bak` / `settings.toml.bak` |


> **Примечание:** Вы можете детально настраивать коэффициенты высыхания, пороговые температуры, объемы осадков и параметры сезонов в файле `settings.toml`, пока приложение закрыто. Подробное описание математической модели полива находится в системном руководстве: введите `man kiri` после установки.

## План развития (Roadmap)

- [ ] **Уведомления на рабочем столе:** Интеграция с нативными системными демонами уведомлений (`notify-send` на Linux, `osascript` на macOS) для отправки алертов на рабочий стол, когда растения хотят пить.
- [ ] **Утилиты импорта/экспорта:** Быстрый дамп и восстановление резервных копий одной командой (например, `kiri --import backup.json`).
- [ ] **Официальная дистрибуция (Пакетные менеджеры):** 
  - [ ] Создание формул для **Homebrew** `brew install freislot/tap/kiri`) для бесшовной установки на macOS. 
  - [ ] Публикация пакетов в **AUR** `kiri` / `kiri-bin`) для пользователей Arch Linux.
  - [ ] Настройка персонального **APT-репозитория** для пользователей Debian/Ubuntu, чтобы обновление шло через стандартные `apt update && apt install kiri`.
- [ ] **Кастомизация тем оформления:** Поддержка пользовательских цветовых схем и встроенное переключение популярных тем (Gruvbox, Catppuccin, Nord и т.д.).
- [ ] **Синхронизация между устройствами:** Автоматическая безопасная и зашифрованная синхронизация состояния между домашним и рабочим компьютерами через приватный Git-репозиторий или менеджер дотфайлов.
- [ ] **Расширяемая система плагинов:** Возможность писать кастомные хуки на Go для получения температуры/влажности с реальных локальных датчиков (вместо погодного API) или управления умным домом (например, автоматическое включение помп на базе ESP32/Arduino).
